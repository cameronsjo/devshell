### Added

- Initial release
- Ubuntu 24.04 base with hardened sshd
- Shell tooling: zsh, tmux, fzf, ripgrep, fd-find, bat, delta
- Dev tooling: git, gh CLI, jq, curl, wget, htop
- Runtime management via mise (Node.js, Python, Go)
- Claude Code (npm global install)
- Infrastructure tools: Docker CLI, sops, age
- Python tooling: uv
- Configurable UID/GID via PUID/PGID environment variables
- Persistent home volume support
- Docker socket passthrough
